

# Automação de Testes com Robot Framework – Avançado

https://www.udemy.com/course/automacao-de-testes-com-robot-framework-avancado/

Repositório dedicado aos estudos e práticas do curso de Automação de Testes com Robot Framework (nível avançado), incluindo automação Web, API, execução em Docker e Integração Contínua.

---

## Objetivo

Reunir exemplos e anotações que permitem revisar rapidamente os principais conceitos avançados do Robot Framework, com foco em:

- Estruturação profissional de suites de teste  
- Uso de bibliotecas nativas e customizadas  
- Automação Web e API avançada  
- Execução dos testes via Docker  
- Pipelines de CI/CD  

---

## Conteúdo Estudado

### 1. Bibliotecas Nativas e Auxiliares
- DateTime  
- OperatingSystem  
- String  
- Process  
- DatabaseLibrary (PostgreSQL)  
- Bibliotecas auxiliares  
- Custom Libraries (Python)  

### 2. Web Testing Avançado
- Page Objects e Resource Files  
- Estratégias avançadas de sincronização (waits)  
- Manipulação de elementos complexos  
- Logs, screenshots e relatórios  
- Boas práticas para testes Web estáveis e robustos  

### 3. API Testing Avançado
- Uso avançado do RequestsLibrary  
- Autenticação, headers, parâmetros e payload dinâmico  
- Validação de JSON, status e contrato  
- Encadeamento de requests  
- Uso de variáveis de ambiente, dados externos e secrets  

### 4. Execução com Docker
- Criação de Dockerfile para testes  
- Execução dos testes em containers  
- Montagem de volumes para relatórios  
- Execução paralela com Docker  

### 5. Integração Contínua (CI/CD)
- Pipelines 
- Execução automática a cada push  

## Tecnologias Utilizadas

- Robot Framework  
- Python  
- SeleniumLibrary  
- RequestsLibrary  
- PostgreSQL  
- Docker
